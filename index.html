P R O Y E C T O S

Movimiento Browniano
A partir del "lazamiento de una moneda" un agente se moverá hacia arriba o hacia abajo. El agente se detendrá al llegar a cierta posición (en este caso 10 o -10).
Problema de las Reinas
En un tablero de ajedrez, son colocadas 8 reinas.
El objetivo es realizar una cantidad mínima de movimientos para que ninguna reina pueda comerse a otra.
Palmer Penguins
Un dataset contiene información relevante para clasificar tres tipos de pingüinos.
El objetivo es preprocesar los datos y escoger el mejor método de clasificación dadas las métricas.
